# todo_app

这是一个用于练习的 Flutter 入门项目，实现了一个简单的待办事项应用。

## 功能特点

- 添加新任务
- 删除任务
- 按分类查看任务
- 点击查看详细

## 技术栈

- Flutter SDK
- Dart 语言
- signals_flutter 状态管理
- shared_preferences 本地数据存储

## 运行项目

1. 确保已安装 Flutter SDK
2. 安装依赖：

```bash
flutter pub get
```

3. 运行项目：

```bash
flutter run
```
